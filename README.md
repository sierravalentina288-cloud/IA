# Inteligencia Artificial
Actividad 2 - Búsqueda y sistemas basados en reglas

# Descripción del Proyecto
El sistema implementa un agente inteligente resolutor de problemas enfocado en la optimización de trayectos viales dentro del municipio de Zipaquirá. Utiliza una base de conocimiento estructurada mediante grafos y aplica el Algoritmo A* para calcular la ruta más eficiente entre una estación y otra estación final.

#Tecnologías Utilizadas
* Lenguaje: Python
